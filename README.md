DotNetCoreMvc-Test
=================

## 用途

以 .NET Core 2.2 MVC + Service 架構，試做僅有登入功能的實驗性 side project 。
因此 project 為粗略設計的架構，且功能沒有多到能體現此架構低耦合性的優點，所以僅供參考指教。
